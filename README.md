# Chemist-Database
The project focuses on managing different types of medication, inventory tracking, sales analysis, and expiry-date monitoring. It supports statistical reporting such as fast-moving medications, low-stock alerts, and sales trends, and includes relational constraints, database views, and triggers for identifying medicines that are nearing expiry.
